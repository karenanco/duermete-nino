# Fixtures - Duérmete Niño

Este directorio contiene los datos iniciales (fixtures) para poblar la base de datos.

## Formato

Los archivos deben estar en formato JSON o YAML, compatibles con `python manage.py loaddata`.

## Uso

```bash
python manage.py loaddata duermete_nino/fixtures/arquetipos.json
python manage.py loaddata duermete_nino/fixtures/paises.json
python manage.py loaddata duermete_nino/fixtures/seres_folcloricos.json
```
