# Plan de Progreso y Futuras Implementaciones

## ✅ Completado (Fase Inicial)

- [x] Modelos Django (Arquetipo, Pais, SerFolclorico)
- [x] Vistas basadas en funciones (FBVs)
- [x] Endpoints API REST (JSON)
- [x] Frontend con Tailwind CSS dark mode
- [x] Catálogo de 100 seres folclóricos en fixtures
- [x] Panel de administración Django
- [x] Configuración de despliegue en Render

## 🔜 Próximos Pasos (Corto Plazo)

### Mapa 3D con WebGL/Three.js
Transición del mapa SVG/placeholder actual a un globo terráqueo tridimensional interactivo utilizando Three.js o Globe.GL, con países cliqueables y rotación suave.

### Precarga Automatizada
Script de migración de datos mejorado para garantizar que los 100 seres se desplieguen sin intervención manual en producción.

### Filtros por Arquetipo
Sistema de filtros en el frontend que permita resaltar en el mapa solo los países que tienen seres de un arquetipo específico.

## 🚀 Futuro (Mediano/Largo Plazo)

### Módulo de Audio
Incorporación de efectos de sonido ambientales (viento, nanas distorsionadas) al seleccionar seres de arquetipos específicos.

### Sección de Aportes Comunitarios
Formulario para que usuarios sugieran nuevos mitos y leyendas de su región, con flujo de revisión y aprobación.

### Mapa de Calor Global
Visualización de densidad de seres por región, mostrando las concentraciones de folclore infantil por arquetipo.

### i18n / Traducciones
Soporte multilingüe (inglés, portugués, francés, alemán, japonés) para alcanzar una audiencia global.
